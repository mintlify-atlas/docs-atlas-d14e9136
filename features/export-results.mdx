## Overview

Export Results allows you to download sentiment analysis data in CSV format for further analysis, reporting, or archival purposes. The API accepts analysis results and returns a downloadable CSV file.

## API Endpoint

```
POST /sentiment/export-csv
```

### Authentication

All requests require a Bearer token in the Authorization header:

```bash
Authorization: Bearer {your_token}
```

## Request Format

The endpoint accepts an array of sentiment analysis results:

```typescript
interface AnalisisResultado {
  texto: string;
  prevision: 'POSITIVO' | 'NEGATIVO';
  probabilidad: number;
}
```

### Request Body Example

```json
[
  {
    "texto": "Excellent product quality!",
    "prevision": "POSITIVO",
    "probabilidad": 0.9456
  },
  {
    "texto": "Very disappointed with the service.",
    "prevision": "NEGATIVO",
    "probabilidad": 0.8823
  }
]
```

### Headers

```json
{
  "Authorization": "Bearer {your_token}",
  "Content-Type": "application/json"
}
```

## Response Format

The API returns a CSV file as a binary blob with the following characteristics:

- **Content-Type**: `text/csv` or `application/octet-stream`
- **Format**: UTF-8 encoded CSV
- **Columns**: Typically includes texto, prevision, probabilidad

## Code Examples

<CodeGroup>

```typescript TypeScript/React
const handleExportCSV = async (
  resultados: AnalisisResultado[],
  token: string
) => {
  if (!token) {
    throw new Error("Sesión expirada. Por favor, inicia sesión nuevamente.");
  }

  try {
    const response = await fetch(
      `${process.env.NEXT_PUBLIC_API_URL}/sentiment/export-csv`,
      {
        method: "POST",
        headers: { 
          "Authorization": `Bearer ${token}`,
          "Content-Type": "application/json"
        },
        body: JSON.stringify(resultados),
      }
    );

    if (!response.ok) {
      throw new Error(`Error ${response.status}: No se pudo generar el CSV`);
    }

    // Convert response to Blob
    const blob = await response.blob();
    
    // Create temporary URL for download
    const url = window.URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = "resultados_sentimiento.csv";
    document.body.appendChild(a);
    a.click();
    
    // Cleanup
    a.remove();
    window.URL.revokeObjectURL(url);
    
  } catch (err: any) {
    if (err.message === "Failed to fetch") {
      throw new Error("No se pudo conectar con el servidor.");
    }
    throw err;
  }
};
```

```python Python
import requests

def export_csv(resultados: list, token: str, output_file: str):
    url = f"{API_URL}/sentiment/export-csv"
    
    headers = {
        "Authorization": f"Bearer {token}",
        "Content-Type": "application/json"
    }
    
    response = requests.post(url, json=resultados, headers=headers)
    response.raise_for_status()
    
    # Save CSV file
    with open(output_file, 'wb') as f:
        f.write(response.content)
    
    return output_file
```

## Client-Side CSV Generation (Alternative)

<Info>
For statistics data, you can also generate CSV files client-side without making an API call.
</Info>

This approach is useful when exporting data from the statistics dashboard:

```typescript
const handleExportCSV = (dataResponse: SentimentApiResponse) => {
  if (!dataResponse || !dataResponse.content) {
    console.error("No hay datos para exportar");
    return;
  }

  try {
    // Define headers and metadata
    const headers = ["ID", "Comentario", "Prevision", "Probabilidad", "Fecha Registro"];
    
    // Create summary section
    let csvContent = `\nRESUMEN DE ANALISIS DE LOS ULTIMOS ${dataResponse.total_en_pagina} COMENTARIOS\n\n`;
    csvContent += `Total en pagina,${dataResponse.total_en_pagina}\n`;
    csvContent += `Positivos,${dataResponse.positivos}\n`;
    csvContent += `Negativos,${dataResponse.negativos}\n\n\n`;
    
    // Add table headers
    csvContent += headers.join(",") + "\n";

    // Map data rows
    const rows = dataResponse.content.map((item) => {
      // Clean text: remove newlines and escape quotes
      const comentarioLimpio = item.comentario
        .replace(/\n/g, " ")
        .replace(/"/g, '""');
      
      return [
        item.id,
        `"${comentarioLimpio}"`,
        item.prevision,
        item.probabilidad,
        `"${item.fecharegistro}"`
      ].join(",");
    });

    csvContent += rows.join("\n");

    // Create Blob with UTF-8 BOM for proper encoding
    const blob = new Blob(["\ufeff" + csvContent], { 
      type: "text/csv;charset=utf-8;" 
    });
    
    const url = URL.createObjectURL(blob);
    const link = document.createElement("a");
    
    link.setAttribute("href", url);
    link.setAttribute("download", `reporte_sentimientos_${new Date().getTime()}.csv`);
    link.style.visibility = 'hidden';
    
    document.body.appendChild(link);
    link.click();
    document.body.removeChild(link);

  } catch (err) {
    console.error("Error al generar CSV:", err);
  }
};
```

## CSV Formatting Best Practices

<Steps>
  <Step title="UTF-8 Encoding">
    Include UTF-8 BOM (`\ufeff`) at the start of the file to ensure proper character encoding
  </Step>
  <Step title="Escape Special Characters">
    Wrap fields containing commas or quotes in double quotes and escape internal quotes by doubling them
  </Step>
  <Step title="Clean Text Data">
    Remove newlines and normalize whitespace in text fields to prevent CSV parsing issues
  </Step>
  <Step title="Timestamps">
    Include timestamps in a sortable format (ISO 8601 or similar)
  </Step>
</Steps>

</Note>

### Common Error Scenarios

| Error | Cause | Solution |
|-------|-------|----------|
| 401 Unauthorized | Invalid or expired token | Re-authenticate the user |
| 400 Bad Request | Invalid data format | Validate data structure before sending |
| 500 Server Error | Server processing error | Retry or contact support |
| Failed to fetch | Network connectivity issue | Check connection and retry |

## Advanced: Custom CSV Formatting

You can customize the CSV output format:

```typescript
const formatProbability = (prob: number) => `${(prob * 100).toFixed(2)}%`;

const createCustomCSV = (data: AnalisisResultado[]) => {
  const headers = ["Text", "Sentiment", "Confidence", "Category"];
  const rows = data.map(item => [
    `"${item.texto.replace(/"/g, '""')}"`,
    item.prevision,
    formatProbability(item.probabilidad),
    item.prevision === 'POSITIVO' ? 'Satisfied' : 'Unsatisfied'
  ]);
  
  return [
    headers.join(','),
    ...rows.map(row => row.join(','))
  ].join('\n');
};
```
